## README

- Ruby version
  - Ruby 2.3.1
  - Rails 5.0

- Database creation
  - terminal: rails db:setup

- Database initialization
  - terminal: rails db:seed

- Run web-app
  - terminal: rails s
  - go in your browser to: localhost:3000
